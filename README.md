# Generador de StringArt
Generador de arte de cuerdas basado en la web

Este proyecto se basa en gran medida en el trabajo del usuario de reddit /u/kmmeerts de su publicación: https://www.reddit.com/r/DIY/comments/au0ilz/made_a_string_art_portrait_out_of_a_continuous_2/ 
Fork of halfmonty's work at https://github.com/halfmonty/StringArtGenerator

Para usar el generador de círculos original, vaya a: https://halfmonty.github.io/StringArtGenerator/

Esta versión que incluye un generador rectangular está disponible en https://greyphilosophy.github.io/StringArtGenerator/, con una severa limitación de que forzará cada imagen en una forma cuadrada. Si quieres que un rectángulo funcione, primero tienes que estirar y cuadrar tu imagen. Con el futuro trabajo en lienzo, esta limitación puede superarse algún día.
